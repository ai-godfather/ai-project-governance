# Agents Adapters

This directory contains AGENTS.md templates for different project types.

## Available Templates

| Template | Use Case |
|----------|----------|
| `base.AGENTS.md` | Universal base template |
| `typescript.AGENTS.md` | TypeScript projects |
| `python.AGENTS.md` | Python projects (coming soon) |

## Usage

1. Copy the appropriate template to your project root
2. Rename to `AGENTS.md`
3. Replace `{{PLACEHOLDERS}}` with project values
4. Customize sections as needed

## Placeholders

| Placeholder | Description |
|-------------|-------------|
| `{{PROJECT_NAME}}` | Your project name |
| `{{DATE}}` | Current date |
| `{{FRAMEWORK}}` | Framework (Express, Next.js, etc.) |
| `{{STACK}}` | Full tech stack description |

## Customization

Each template includes:
- Quick start guide
- Role definitions
- Anti-patterns
- Quick commands

Add project-specific sections as needed.
