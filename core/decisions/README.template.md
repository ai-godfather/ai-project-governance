# Decisions Directory

> **Project**: {{PROJECT_NAME}}

---

## Purpose

This directory tracks all decision-making artifacts:

- **FEATURE_INDEX.md** — All feature plans index
- **TASK_INDEX.md** — Task statistics and status
- **CHANGELOG_INDEX.md** — Changelog entries overview
- **SNAPSHOT_INDEX.md** — System state snapshots

---

## How to Use

1. **Feature Plans**: Reference `FEATURE_INDEX.md` for all active/completed plans
2. **Task Status**: Check `TASK_INDEX.md` for current task progress
3. **History**: Review `CHANGELOG_INDEX.md` for change history

---

## Update Cadence

- **FEATURE_INDEX**: Updated when new plan created/completed
- **TASK_INDEX**: Updated when task status changes
- **CHANGELOG_INDEX**: Updated daily (if changes occurred)
- **SNAPSHOT_INDEX**: Updated after significant system state changes
